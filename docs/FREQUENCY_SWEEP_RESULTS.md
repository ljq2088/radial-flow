## 物理分析

### 1. 低频行为（ω < 0.5）

**特征**：|A/B| ≈ 1（完全反射）

**物理解释**：
- 波长 λ = 2π/ω 很长（λ > 12）
- 势垒对长波长波有效阻挡
- 波无法穿透势垒，几乎完全反射

**收敛特性**：
- ω = 0.1 需要 r/r₀ = 305 才能收敛（波长最长，需要最远距离）
- ω = 0.2, 0.3 在 r/r₀ = 200 收敛

### 2. 中频行为（0.5 < ω < 1.0）

**特征**：|A/B| 从 0.77 降到 0.06

**物理解释**：
- 波长适中（6 < λ < 12）
- 部分波穿透势垒，部分反射
- 频率越高，透射越多

**收敛特性**：
- 在 r/r₀ = 200 快速收敛

### 3. 高频行为（ω ≥ 1.0）

**特征**：|A/B| ≪ 1（几乎完全透射）

**物理解释**：
- 波长短（λ ≤ 6）
- 几何光学极限：波直接穿过势垒
- 反射系数很小

**收敛特性**：
- ω = 1.0 需要 r/r₀ = 1100（特殊情况，收敛慢）
- ω = 1.5, 2.0 在 r/r₀ = 200 收敛
- ω = 3.0, 5.0 未收敛（可能需要更大的 n_periods 或更宽松的阈值）

### 4. 反常行为：ω = 2.0

**观察**：|A/B| = 0.261295（比 ω = 1.5 的 0.017227 更大）

**可能原因**：
- 准正规模共振效应
- 势垒的特定频率响应
- 需要进一步物理分析

## 收敛问题分析

### ω = 1.0 的慢收敛

**现象**：需要 r/r₀ = 1100，积分点数 55,209

**原因**：
- 波长 λ = 2π ≈ 6.28
- 检测窗口：5个周期 = 790个点
- 相对震荡下降非常缓慢：5.18 → 2.68 → 1.77 → ... → 0.10

**物理解释**：
- 这个频率可能接近某个特征频率
- 渐近行为建立得很慢
- 需要传播很远才能进入真正的渐近区

### ω = 3.0, 5.0 的不收敛

**现象**：即使在 r/r₀ = 2000 也未收敛

**可能原因**：
1. **检测窗口太小**：
   - ω = 3.0: check_window = 265 点（5个周期）
   - ω = 5.0: check_window = 160 点（5个周期）
   - 可能需要更多周期（如 10 或 20）

2. **数值精度问题**：
   - |A/B| 非常小时，相对误差可能很大
   - 需要更高的数值精度

3. **物理原因**：
   - 高频时可能存在特殊的物理效应
   - 需要更详细的理论分析

## 建议

### 对于 ω = 3.0, 5.0

1. **增加检测周期**：
   ```python
   result = solver.solve(
       r_max=200.0,
       convergence_threshold=0.1,
       n_periods=20,  # 增加到20个周期
       max_r_factor=10.0
   )
   ```

2. **放宽阈值**：
   ```python
   convergence_threshold=0.2  # 20%
   ```

3. **检查数值稳定性**：
   - 减小步长 dr
   - 使用更高阶的积分方法

### 一般建议

1. **低频（ω < 0.5）**：
   - 使用 n_periods = 5-10
   - threshold = 0.01-0.1
   - 可能需要 r_max > 200

2. **中频（0.5 < ω < 2.0）**：
   - 使用 n_periods = 2-5
   - threshold = 0.01
   - r_max = 200 通常足够

3. **高频（ω > 2.0）**：
   - 使用 n_periods = 10-20
   - threshold = 0.1-0.2
   - 可能需要特殊处理
