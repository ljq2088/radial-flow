### 修正 Riccati 方程 c 系数符号错误

**问题：** 完整版本的二阶项系数 c 缺少负号

**修正：** 
```cpp
// 修正前
c = -1.0/r + V/(2.0*Complex(0, 1)*omega);

// 修正后
c = -(-1.0/r + V/(2.0*Complex(0, 1)*omega));
```

**影响：**
- 修正前：|A/B| = 0.390（错误，比简化版本增大 28%）
- 修正后：|A/B| = 0.235（正确，比简化版本减小 23%）

**物理意义：** 次领头项 f'/r 使势垒增高，反射系数减小，符合物理预期。
