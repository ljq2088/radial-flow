# 重要更新说明

## 日期：2026-02-05

### 🔧 修正内容

发现并修正了完整版本（use_subleading_term=True）中 Riccati 方程二阶项系数 c 的符号错误。

**错误代码：**
```cpp
// src/radial_flow.cpp, line 45
c = -1.0/r + V/(2.0*Complex(0, 1)*omega);  // 缺少负号
```

**修正后：**
```cpp
c = -(-1.0/r + V/(2.0*Complex(0, 1)*omega));  // 添加负号
```

### 🎯 物理意义

**修正前（错误）：**
- 完整版本的反射系数比简化版本**增大** 28%
- 不符合物理预期（次领头项应使势垒增高，反射减小）

**修正后（正确）：**
- 完整版本的反射系数比简化版本**减小** 23%
- 符合物理预期：次领头项 f'/r 使势垒增高，导致反射系数减小

### 🔍 技术细节

Riccati 方程形式：
```
dσ/dr = a + bσ + cσ²
```

对于完整版本（去掉 a 系数的 -1/r 项）：
- a = -V/(2iω)
- b = 2iω/f + V/(iω)
- c = **-**(-1/r + V/(2iω))  ← 必须有负号

这个负号确保了 Riccati 方程的正确形式，使得数值解符合物理预期。
