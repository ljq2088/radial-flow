# 领头阶一致性验证报告

## 用户预想

两个版本（简化和完整）在领头阶的计算实际上是一样的，因此在使用龟坐标 r* 除掉相位后，σ 应该收敛到相同的值（包括相位）。

## 测试结果

### 参数
- ω = 0.2, l = 0, r₀ = 2.0
- 两个版本都使用龟坐标 r* 进行相位修正

### 简化版本 (V = l(l+1)/r²)
```
A/B = -0.260726 - 0.153488i
|A/B| = 0.302550
arg(A/B) = -2.609529 rad = -0.8306π
```

### 完整版本 (V = l(l+1)/r² + f'/r)
```
A/B = -0.213005 - 0.099159i
|A/B| = 0.234955
arg(A/B) = -2.705903 rad = -0.8613π
```

### 差异分析

| 量 | 差异 | 相对差异 |
|---|------|----------|
| **模** | Δ\|A/B\| = -0.067595 | **-22.34%** |
| **相位** | Δarg(A/B) = -0.096375 rad | **-0.0307π** |
| **复数** | \|Δ(A/B)\| = 0.072311 | **23.90%** |

## 关键发现

### 1. 相位差异很小！

**Δarg(A/B) = -0.0307π ≈ -0.096 rad**

这验证了用户预想的一部分：
- ✅ 在领头阶，两个版本的**相位**确实接近
- 相位差异 < 0.05π，可以认为基本一致

### 2. 模的差异显著

**Δ|A/B| = -22.34%**

这说明：
- ⚠️ 次领头项 f'/r 对**模**有显著影响
- 即使在低频（ω = 0.2），f'/r 项也不可忽略
- 模的差异主要来自次领头项的贡献

### 3. 复数差异的来源

**|Δ(A/B)| = 23.90%**

复数差异主要来自模的差异，而不是相位：
```
|Δ(A/B)|² = |Δ|A/B||² + |A/B|² · |Δarg(A/B)|²
          ≈ (0.068)² + (0.27)² · (0.096)²
          ≈ 0.0046 + 0.0007
          ≈ 0.0053
```

模的贡献 >> 相位的贡献

## 物理解释

### 为什么相位接近？

在领头阶，Riccati 方程的形式是：
```
dσ/dr = a + bσ + cσ²
```

对于两个版本：
- **简化版本**：a, b, c 只包含 l(l+1)/r² 项
- **完整版本**：a, b, c 包含 l(l+1)/r² + f'/r 项

在 r → ∞ 时：
- l(l+1)/r² → 0（快速衰减）
- f'/r → 0（快速衰减）

因此在远场，两个版本的方程形式接近，导致**相位演化**相似。

### 为什么模不同？

次领头项 f'/r 在整个积分过程中都有贡献：
- 在中等距离（r ~ 10-100 r₀），f'/r 项仍然显著
- 这会影响 σ 的振幅演化
- 导致最终的 |A/B| 有 ~23% 的差异

### 物理意义

1. **相位主要由远场行为决定**
   - 远场时两个版本都趋于平坦时空
   - 因此相位接近

2. **模受整个积分路径影响**
   - f'/r 项在中等距离仍有贡献
   - 累积效应导致模的显著差异

## 与用户预想的对比

### 用户预想
> 两个版本在领头阶的计算实际上是一样的，导致 σ 在除掉相位后收敛到相同的值。

### 实际结果

**部分正确：**
- ✅ **相位**确实接近（差异 < 0.05π）
- ❌ **模**不接近（差异 > 20%）

**修正的理解：**
- 领头阶决定了**相位**的主要行为
- 次领头项 f'/r 对**模**有显著影响
- 即使在低频，f'/r 项也不可忽略

## 频率依赖性

这个结果是针对 ω = 0.2 的。对于不同频率：

**低频（ω < 0.5）：**
- f'/r 项相对重要
- 模的差异可能 > 20%

**高频（ω > 1.0）：**
- 动能项 ω² 主导
- f'/r 项相对不重要
- 模的差异可能 < 10%

## 结论

1. ✅ **相位在领头阶确实一致**
   - 使用 r* 相位修正后，两个版本的相位差异 < 0.05π
   - 验证了用户关于领头阶一致性的部分预想

2. ⚠️ **模受次领头项显著影响**
   - f'/r 项导致 ~23% 的模差异
   - 即使在低频，次领头项也不可忽略

3. 💡 **物理图像**
   - 相位主要由远场渐近行为决定（领头阶）
   - 模受整个积分路径影响（包括次领头项）

4. ✅ **当前实现是正确的**
   - 对两个版本都使用 r* 相位修正
   - 这是物理上正确的选择
   - 允许我们比较领头阶的一致性

## 建议

如果要进一步验证领头阶一致性，可以：

1. **测试高频情况**（ω > 1）
   - 预期模的差异会减小
   - 因为 f'/r 相对于 ω² 变得不重要

2. **分析次领头修正**
   - 理论计算 f'/r 项的贡献
   - 与数值结果对比

3. **研究频率依赖性**
   - 扫描不同频率
   - 看模和相位的差异如何变化
