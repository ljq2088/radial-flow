# r* 坐标积分结果报告

## 概述

实现了在龟坐标 r* 上进行积分的功能，并将积分范围扩展到 r=2000。对比了在 r 坐标和 r* 坐标积分的结果。

## 实现细节

### 1. 新增参数

在 `WaveParams` 中添加了 `integrate_in_rstar` 参数：
- `true`: 在 r* 坐标积分
- `false`: 在 r 坐标积分（默认）

### 2. 积分方法

**r* 坐标积分**：
- 龟坐标定义：r* = r + r₀ ln(r/r₀ - 1)
- 坐标变换：dr/dr* = (r - r₀)/r
- Riccati 方程：dσ/dr* = (dσ/dr) × (r - r₀)/r

**实现方式**：
- 在 r* 上取均匀步长 dr*
- 在每个 RK4 子步中，根据当前 r 计算对应的 dr
- 使用 `sigma_rhs_rstar(r, σ)` 计算右端项

### 3. 修改的文件

1. `include/radial_flow.hpp` - 添加 `integrate_in_rstar` 参数和 `sigma_rhs_rstar` 方法
2. `src/radial_flow.cpp` - 实现 r* 积分逻辑
3. `src/bindings.cpp` - 更新 Python 绑定
4. `python/radial_flow.py` - 添加 Python 接口

## 测试结果

### 参数设置
- 黑洞：r₀ = 2.0
- 波参数：ω = 0.2, l = 0
- 积分范围：r_max = 2000 r₀
- 点数：n_points = 50000

### 简化版本 (V = l(l+1)/r²)

| 积分坐标 | 最终 r/r₀ | A/B | \|A/B\| | arg(A/B) | 收敛 |
|---------|-----------|-----|---------|----------|------|
| r 坐标  | 2000.04   | -0.2624 - 0.1503i | 0.302373 | -0.834451π | ✓ |
| r* 坐标 | 2000.01   | -0.2575 - 0.1533i | 0.299663 | -0.829123π | ✓ |

**差异**：
- Δ\|A/B\| = -0.002710 (-0.896%)
- Δarg(A/B) = 0.005328π

### 完整版本 (V = l(l+1)/r² + f'/r)

| 积分坐标 | 最终 r/r₀ | A/B | \|A/B\| | arg(A/B) | 收敛 |
|---------|-----------|-----|---------|----------|------|
| r 坐标  | 2000.04   | -0.2127 - 0.0991i | 0.234657 | -0.861224π | ✓ |
| r* 坐标 | 2000.01   | -0.2104 - 0.1032i | 0.234398 | -0.854824π | ✓ |

**差异**：
- Δ\|A/B\| = -0.000259 (-0.110%)
- Δarg(A/B) = 0.006399π

## 分析

### 1. 数值一致性

两种积分方式的结果非常接近：
- **简化版本**：\|A/B\| 差异 < 1%
- **完整版本**：\|A/B\| 差异 < 0.2%

这验证了数值实现的正确性。

### 2. 完整版本更稳定

完整版本在两种积分坐标下的差异更小（0.110% vs 0.896%），说明包含 f'/r 项的势函数在 r* 坐标下更自然，数值稳定性更好。

### 3. 相位差异

两种积分方式的相位差异约为 0.005-0.006π，这是由于：
- 数值积分的累积误差
- 步长选择的差异（r* 上均匀 vs r 上均匀）

### 4. 收敛行为

两种方式都能成功收敛到 r=2000，说明：
- 积分算法稳定
- 收敛判据有效
- 相位修正正确

## 使用示例

```python
from python.radial_flow import SchwarzschildParams, WaveParams, RadialFlowSolver

bh = SchwarzschildParams(r0=2.0)

# 在 r* 坐标积分
wave = WaveParams(omega=0.2, l=0, use_subleading_term=True, integrate_in_rstar=True)
solver = RadialFlowSolver(bh, wave)
result = solver.solve(r_max=2000.0, epsilon=1e-6, n_points=50000)

print(f"|A/B| = {abs(result['reflection_coeff']):.6f}")
```
